# Prompts de Geração de Cartas (Pixel Art)

Esses prompts foram desenhados para manter o estilo visual exato da imagem de referência original. Use o mesmo padrão (fundo escuro, estilo 64x64 isolado e temática pixel art fantasia) caso use Midjourney, DALL-E ou ferramentas parecidas.

O formato padrão é:
`Pixel art skill icon for a fantasy game: [DESCRIÇÃO]. Dark background, isolated icon, 64x64 style.`
